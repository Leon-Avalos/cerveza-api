# Api de cervezas

## Descripcion


Con esta API Rest podras consultar informacion basica sobre la cerveza tal como:

  ..*Nombre comercial
  ..*Tipo de cerveza
  ..*Fabricante
  ..*Porcentaje de alcohol
  ..*Presentacion en mililitros



## Requerimientos

|   Dependencia      | Version     |
|--------------------|:-----------:|
| Visual Studio Code  |Cualquiera  |
| .NET Core SDK       |2.2 o >     |
| C# para Visual Code |1.17.1 o >  |
